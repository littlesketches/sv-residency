SV Recovered Resources Market Bulletin graphics index

This page contains links to HTML/SVG/JS based data visualisations that (will be) embedded into various sections of online version of the SV Recovered Resources Market Bulletin. The aim for each of these visualisations is that they will automatically update when data is updated in the supporting data tables. Because there a need to archive editions of the Market Bulletin with charts displaying relavent (historical) dates, each ot these apps includes the ability to request dates using a query string (detailed below for each app/link)

All graphics are designed as small single page data-driven apps that are embeddable on the SV wesbite using the "Advanced Data Visualisation" entry, with D3 selected as the "Entry Type" (in the SV website CMS). This option allows for HTML pages (i.e. like those linked) below to be embedded in iframes on the SV website content pages. It should be noted that for embedded content to properly size in the CMS the iframe resizer library is required to be included in the embedded app (and has been included as a CDN link in al of the links below)

The linked dataset supporting these visualisations can be found and updated here. These tables should be replaced with connectors to SV operated cloud database services (when available) - the current data connection is made to a text file database (tab seperated values) provided by the Google Sheets service. These connections are supported by the Papa Parse 5 library.

The GitHub repository (i.e. code) for these visualisations and apps can be found here (note: this will be moved to a SV data team repo shortly).

About the visualisations

There are five separate code bases that support these visualisations:

  1. Kerbside collection flows: is an application that supports the kerbside flows graphic only. It is connected to the 'data_mrfOutput' table
  2. Time series charts: is an application that supports all interactive line, area and layer (stacked area) charts. These charts are individually configured (from JS objects located directly in their HTML page) to connect to either the 'data_mrfOutput' or 'data_materialsVicExport' tables
  3. Materials dashboard: is an application that supports the 'flow, price and exports' dashboards and individual components used as 'performance tables'. These are connected to either or all of the 'data_mrfOutput', 'data_materialsVicExport' and 'data_commodityValues' tables
  4. Value chain price: is a small application that supports the value chain pricing diagrams. It's geometry (i.e. positioning of node and price labels) is configured from an JS object (editable from within its main.js code) and is connected to the 'data_commodityValues' tables
  5. Market capacity tables: is a small application that produces the colour-coded end market capacity tables (as colour-coded tables are not currently supported by the CMS). Its inputs (including date for publication) are contained in the 'data_marketCapacity' tables

Further reference documentation will be added to each applications github repo page (soon).
